## Getting Started Roadmap

Follow these steps to launch your first project with Pepperstars.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at https://app.pepperstars.dev and verify your email.
  </Step>
  <Step title="Install SDK" icon="download">
    Choose your preferred language and install the SDK.

    <CodeGroup tabs="Node.js,Python">
      ```bash
      npm install pepperstars-sdk
      ```
      ```bash
      pip install pepperstars-sdk
      ```
    </CodeGroup>
  </Step>
  <Step title="Initialize Project" icon="settings">
    Set up your first project with a simple configuration.

    ```javascript
    import { Pepperstars } from 'pepperstars-sdk';

    const client = new Pepperstars({ apiKey: 'YOUR_API_KEY' });
    await client.init();
    ```
  </Step>
  <Step title="Deploy" icon="rocket">
    Push your code and deploy live.

    <Callout kind="tip">
      Use `client.deploy()` for one-click production deploys.
    </Callout>
  </Step>
</Steps>
